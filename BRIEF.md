# SUMMARY

A single internal dashboard for a logistics company 'FastForward Logistics'.  For use in meetings to see how well business is running.  Focused on 4 KPIs -  shipment volume, on-time delivery rates, regional performance, and open exceptions.  It also provides a simple analysis feature to help make sense of the data. 

## DESIGN

- Single page application for a year's worth of logistics data.

- Header section with a dashboard title, and a Performance Analysis button which provides a simple analysis of the data present in the dashboard. 

- 4 cards along the top showing the KPIs - shipment volume, on-time delivery, rates, regional performance, and open exceptions.

- Below that graphs for each of the 4 KPIs: shipment volume, on-time delivery rates, regional performance, and open exceptions.

- Make the design responsive to changing browser window sizes.

- Make the aesthetic minimal and clean.

## DATA

- create a set of data for each of the KPIs, to create the graphs from. 

- make shipment volume gradually increase, month over month

- make on-time delivery rates gradually decrease with some variance, related to increases in shipment volumne

- make open exceptions gradually increase with some variance, related to increases in shipment volume

- make regional performance vary, loosely correlated with shipment rates

## TECH

- only html, css and javascript, no frameworks
